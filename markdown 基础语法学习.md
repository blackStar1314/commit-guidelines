# 标题
## 语法

- 在字和短语前添加数字符号`#`创建标题。
- 标题分6层
- 例子
  1. # 标题1
  2. ## 标题2
  3. ### 标题3
  4. #### 标题4
  5. ##### 标题5
  6. ###### 标题6

## 备用语法

- 在文本下面添加`==`或`--`
- 标题分2层
- 例子
  
标题1
==
标题2
--

## 建议写法

```
Good : # 标题
Bad : #标题
```

# 段落
## 语法

- 使用空行分隔一行或多行文本
- 例子
```
第一行文本

第二行文本
```

## 建议写法

```
Good
不要缩进
Bad
    缩进
Good
保持行对齐
Bad
 不对齐
```

# 换行
## 语法

- 以两个或多个空格结束一行，然后键入回车
- 例子
    第一行  
    第二行
    第三行

## 建议写法
```
Good
第一行
第二行
Bad
第一行\
第二行
```

# 强调
## 加粗
### 语法

- 在单词或短语之前和之后添加两个`**`或`__`
- 例子
我**非常**喜欢武侠小说
我__非常__喜欢武侠小说

### 推荐写法
```
Good
我**非常**喜欢武侠小说
Bad
我__非常__喜欢武侠小说
```

## 斜体
### 语法

- 在单词或短语之前和之后添加一个`*`或`_`
- 例子
我*非常*喜欢武侠小说
我 _非常_ 喜欢武侠小说

### 推荐写法
```
Good
我*非常*喜欢武侠小说
Bad
我 _非常_ 喜欢武侠小说
```

## 斜体加粗
### 语法
- 在单词或短语之前和之后添加三个`***`或`___`
- 例子
我 ***非常*** 喜欢武侠小说
我 ___非常___ 喜欢武侠小说

### 推荐写法

```
Good
我 ***非常*** 喜欢武侠小说
Bad
我 ___非常___ 喜欢武侠小说
```

# 块引用
## 非嵌套块引用
### 语法

- 在段落前添加`>`
- 例子


> 这是块引用内容
> 
> 这是块引用 内容

## 嵌套块引用
### 语法

- 在需要嵌套的段落前添加`>>`
- 例子
  
> 这是块引用内容
>> 这是嵌套块引用内容
>
> 这是块引用内容

## 块引用中使用其他元素
### 语法

- 在块引用内部引用其他 **markdown 其他语法**，但是 ***不适用所有语法***。
- 例子

> 这是块引用 *内容*
>> 这是块引用 **内容**
> 这是块引用 ***内容***
> - 这是块引用内容
> 1. 这是块引用内容
> # 这是块引用内容
> [百度](https://www.baidu.com)
> # **... ...**

# 列表
## 有序列表
### 语法

- 在行项前添加`数字.`，不要求数字严格递增
- 例子
> 1. itme1
> 2. item2
>
> 1. item3
> 19. item4
>>    1.  subitem1
>>    2.  subitem2
> 20. item5

## 无序列表
### 语法

- 在行项前添加`-`、`*`或`+`
- 例子
- item
- 
* item

+ item
  + item
  +  item

### 添加数字
#### 语法

- 如果需要在行项前添加`数字.`，需要使用`\`转义
- 例子
- 1\. 项
- 项1. 的内容

## 列表中添加元素
### 语法
- 在块引用内部引用其他 **markdown 其他语法**
- 例子
* 行项1
  > 块引用
  ### 标题
  第一段
  第二段
  ##  ... ...
* 行项2

## 推荐写法

```
Good
1. item
2. item
3. item
Bad
1. item
1. item

Good
- item
  - item
- itme
Bad
- item
+ item
* item
```

# 代码
## 语法

- 在单词或短语前后添加 *`* 表示为代码
- 例子
我想把这个`字`标记为代码
`#include <iostream>`

## **`** 转义
- 在单词或短语前添加两个 *``* 转义
- 例子
`fd` 
`` `代码`

- 例子
***

---

___

# 链接
## 标题
### 语法

- 将链接使用`[title](link)`括起来
- 例子
[百度](https://www.baidu.com)

## 快速转换为链接
### 语法

- 将URL或其他地址用`<URL>`括起来
- 例子
<https://www.baidu.com>
<smartneo@outlook.com>


## 格式化链接

- 例子
*[google](https://google.com)*
**[google](https://google.com)**
### [谷歌](https://google.com)

# 图片
## 语法

- 将图片地址用`(PATH/URL "title")`括起来，括号前添加`![ALT文本]`
- 例子
  - 无ALT文本无标题
  ![](https://mdg.imgix.net/assets/images/san-juan-mountains.jpg?auto=format&fit=clip&q=40&w=1080)
  - 有ALT文本有标题
  ![美景!](https://mdg.imgix.net/assets/images/san-juan-mountains.jpg?auto=format&fit=clip&q=40&w=1080 "有标题")
  - 图片添加链接
  [![美景](https://mdg.imgix.net/assets/images/san-juan-mountains.jpg?auto=format&fit=clip&q=40&w=1080)](https://www.google.com)
    
# 转义字符

- 在需要转义的字符前添加`\`
- 可转义字符
  |字符|名称|
  |:----:|:----:|
  |\\|反斜杠|
  |`|反撇点|
  |*|星号|
  |-|短横|
  |{}|花括号|
  |[]|方括号|
  |<>|尖叫括号|
  |()|圆括号|
  |#|井号|
  |+|加号|
  |-|减号|
  |.|点号|
  |!|感叹号|
  |\||竖线|

  ## HTML
